# 🆓 Kostenlos Deployment - Schritt für Schritt

## Was wir verwenden (100% KOSTENLOS):

### 1. Firebase Hosting (Frontend)
- Kostenlos unlimitiert
- Deine App erreichbar unter: `https://mitarbeiterevent-prod.web.app`

### 2. Render.com (Backend/Express Server)
- Kostenlos (Web Service)
- Auto-suspend nach 15min Inaktivität (kein Problem)
- Einfach zu deployen

### 3. Firebase Realtime Database (Daten)
- Kostenlos großes Kontingent
- 1 GB Speicher kostenlos
- 100 gleichzeitige Verbindungen

## Die 5 Schritte zum Deploy:

### SCHRITT 1: Firebase Project erstellen
1. Gehe zu https://console.firebase.google.com
2. Klicke "Create Project"
3. Project-Name: `mitarbeiterevent-prod`
4. ❌ Google Analytics: NEIN (sparen)
5. Bestätige

### SCHRITT 2: Firebase Realtime Database aktivieren
1. Gehe zu "Build" → "Realtime Database" 
2. Klicke "Create Database"
3. Location: Europe (Belgium) - `eur3`
4. Sicherheit: "Start in test mode"
5. Erstelle

### SCHRITT 3: Firebase Konfiguration
1. Project Settings (Zahnrad oben)
2. Kopiere die Config JSON
3. Speichern für später

### SCHRITT 4: Render.com Account
1. Gehe zu https://render.com
2. Sign Up mit GitHub
3. Verbinde dein GitHub Repo

### SCHRITT 5: Hier wird es automatisch
- Du sendest mir die Firebase Config
- Ich konfiguriere alles fertig
- `firebase deploy` und done!
